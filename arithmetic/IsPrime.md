**Problem Description**  

Given a number **A**. Return 1 if **A** is prime and return 0 if not. 

**Note :**   
The value of **A** can cross the Integer range.

    
**Problem Constraints**  

1 <= **A** <= 10<sup>9</sup>

  
**Input Format**  

The first argument is a single integer A.

  
  
**Output Format**  

Return 1 if A is prime else return 0.

  
  
**Example Input**  

Input 1:

```
A = 5
```

Input 2:

```
A = 10
```

  
  
**Example Output**  

Output 1:

```
1
```

Output 2:

```
0
```

  
  
**Example Explanation**  

Explanation 1:

```
5 is a prime number.
```

Explanation 2:

```
10 is not a prime number.
```
